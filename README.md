# CashTrack-Automation-Bank
Projeto para agilizar diariamente a conciliação de extrato bancário através da baixa do OFX, o sistema vai ler linha por linha e  buscar em regras os padrões encontrados no extrato alimentando [ Tipo de Gasto, Categoria, Fornecedor, Forma de Pagamento e Centro de Custo ]. Feito isso a gente salva a linha até estar tudo ok
