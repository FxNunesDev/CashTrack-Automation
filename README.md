## Objetivo

Reduzir trabalho manual, padronizar lançamentos financeiros e acelerar rotinas operacionais através de automação inteligente.

---

## Principais Funcionalidades

✅ Importação automática de arquivos OFX  
✅ Classificação automática de transações  
✅ Regras inteligentes em YAML  
✅ Preenchimento automático de campos  
✅ Tratamento de descrições bancárias  
✅ Redução de erros manuais  
✅ Ganho de velocidade operacional

---

## Tecnologias Utilizadas

- Python
- Selenium
- Playwright (evolução futura)
- YAML
- Pandas
- Automação Web
- ETL

## Como Executar

```bash
git clone https://github.com/FxNunesDev/CashTrack-Automation.git
cd CashTrack-Automation
pip install -r requirements.txt
python selenium_cashtrack_v5.py
```
